# ⚡ I.M. Suplementos | Performance Store

Uma vitrine digital de alta performance desenvolvida para maximizar a conversão de vendas de suplementos esportivos. O projeto combina um design premium focado em UX com uma arquitetura front-end otimizada para velocidade e inteligência de negócios.

## 🚀 Visão Geral e Funcionalidades

* **Catálogo Dinâmico e Estratégico:** Renderização de produtos via manipulação do DOM, com sistema de filtros responsivos (por objetivo, categoria e marca).
* **Conversão Direta (WhatsApp):** Geração de checkout ágil. Os botões de compra criam dinamicamente a mensagem de intenção contendo nome do produto e preço exato, enviando o lead pronto para o fechamento.
* **UI/UX Premium (Mobile-First):** Interface desenvolvida com foco no usuário móvel. Inclui carrossel horizontal contínuo para filtros, efeitos de *Glassmorphism* nos cards de assinatura e hierarquia visual focada em conversão.
* **Business Intelligence (GA4):** Infraestrutura de dados integrada com o Google Analytics 4 para rastreamento de eventos personalizados:
    * `click_comprar_whatsapp`: Monitora os produtos que geram mais cliques.
    * `uso_de_filtro`: Mapeia a intenção de busca do usuário para auxiliar na previsão de estoque.
* **Clean Code & Performance:** Aplicação de boas práticas como `loading="lazy"` para assets visuais, CSS Custom Properties (Variáveis) para design tokens e modularização de funções em JavaScript moderno.

## 🛠️ Tecnologias e Ferramentas

* **HTML5:** Estrutura semântica rigorosa para acessibilidade e SEO.
* **CSS3:** Flexbox, CSS Variables e Media Queries para adaptação fluida em todas as telas.
* **JavaScript (Vanilla ES6+):** Lógica de estado da página e integração de eventos de conversão sem dependência de frameworks pesados.
* **Lucide Icons:** Implementação de ícones SVG dinâmicos e leves.
* **Google Analytics 4:** Coleta de métricas e comportamento do usuário.
